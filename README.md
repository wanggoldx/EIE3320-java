# EIE3320 - Java Programming

## Course Overview

This course teaches object-oriented programming (OOP) using Java. Students learn fundamental OOP concepts and their practical application through Java's rich class library.

## Topics Covered

- Object-oriented programming fundamentals
- Classes, objects, and encapsulation
- Inheritance and polymorphism
- Abstract classes and interfaces
- Collections and generics
- Exception handling
- GUI programming (AWT/Swing)

## Labs

### Lab 1-2: OOP Fundamentals
- Abstract classes
- Interfaces
- Inheritance
- Polymorphism
- Collections (ArrayList)

## Homework

The course includes 8 homework assignments covering:

1. **Homework 1-8**: Progressive exercises covering:
   - Class design and implementation
   - Inheritance hierarchies
   - Interface usage
   - Collection management
   - Exception handling
   - GUI applications

## Learning Objectives

By the end of this course, students will be able to:
1. Design and implement Java classes
2. Apply OOP principles (encapsulation, inheritance, polymorphism)
3. Use Java collections effectively
4. Handle exceptions gracefully
5. Build GUI applications

## Software Requirements

- Java Development Kit (JDK) 8 or higher
- BlueJ (recommended) or any Java IDE (Eclipse, IntelliJ, NetBeans)

## Key Concepts

| Concept | Description |
|---------|-------------|
| Abstract Class | Base class with unimplemented methods |
| Interface | Contract for behavior |
| Inheritance | Class extension relationships |
| Polymorphism | Treating objects uniformly |
| Encapsulation | Data hiding with public methods |
| ArrayList | Dynamic collection storage |

## Course Projects

- **Shape Drawing System**: Demonstrates polymorphism with geometric shapes
- **Collection Management**: ArrayList operations with objects

# Homework 8 - Binary Search Tree

## What it does

A generic Binary Search Tree (BST) implementation with insert, delete, search, and tree traversals (inorder, preorder, postorder). Supports custom comparators and provides an inorder iterator.

## How to run

Compile all files and run:
```bash
javac Tree.java BST.java UseBST.java
java UseBST
```

## Dependencies

- Java (any version with Comparator)
- Tree.java - Tree interface with abstract methods
- BST.java - Binary Search Tree implementation
- UseBST.java - Test driver demonstrating tree operations

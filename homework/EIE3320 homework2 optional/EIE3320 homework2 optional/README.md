# Homework 2 Optional - Rectangle Test

## What it does

Tests the Rectangle class by creating two rectangle objects with given width/height values and printing their area and perimeter.

## How to run

Compile and run with Java:
```bash
javac Rectangle.java testq2.java
java testq2
```

## Dependencies

- Java (any version)
- Rectangle.java (class with getArea() and getPerimeter() methods)

# Homework 1 - Number Triangle

## What it does

Prompts the user to enter a number of lines, then prints a centered triangle pattern of numbers (1, 12, 123, etc.) in a pyramid shape.

## How to run

Compile and run with Java:
```bash
javac question1.java
java question1
```

Enter the number of lines when prompted.

## Dependencies

- Java (any version with Scanner support)
- No external libraries required

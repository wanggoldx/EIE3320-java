# Homework 3 - Course & University Student

## What it does

Tests the Course and UniversityStudent classes. The code creates Course objects with course name and test marks, and UniversityStudent objects with name and enrolled courses, then prints their details.

## How to run

Compile all files and run:
```bash
javac Course.java UniversityStudent.java Test1Q2A.java
java Test1Q2A
```

## Dependencies

- Java (any version)
- Course.java - Course class with courseName and testMarks
- UniversityStudent.java - Student class that holds courses

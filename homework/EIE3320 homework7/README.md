# Homework 7 - Stack & Queue Implementation

## What it does

Implements generic Stack and Queue data structures using ArrayList. Includes test code demonstrating push/pop operations on Stack and enqueue/dequeue on Queue.

## How to run

Compile all files and run:
```bash
javac MyStack.java MyQueue.java TestStackQueue.java
java TestStackQueue
```

## Dependencies

- Java (any version with ArrayList)
- MyStack.java - Generic stack (push, pop, peek, getSize, isEmpty)
- MyQueue.java - Generic queue (enqueue, dequeue, getFront, getSize, isEmpty)
- TestStackQueue.java - Test driver

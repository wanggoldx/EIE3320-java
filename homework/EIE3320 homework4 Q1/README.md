# Homework 4 Q1 - CD & Video Database

## What it does

A database system that stores and manages CD, Video, and VideoGame objects. The TestDatabase creates sample entries and prints all stored items.

## How to run

Compile all files and run:
```bash
javac CD.java Video.java VideoGame.java Database.java TestDatabase.java
java TestDatabase
```

## Dependencies

- Java (any version with ArrayList)
- CD.java - CD class with title, artist, and tracks
- Video.java - Video class with title and director
- VideoGame.java - VideoGame class with title, platform, and genre
- Database.java - Storage for all media types

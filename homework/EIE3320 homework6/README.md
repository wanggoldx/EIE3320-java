# Homework 6 - Shape Drawing GUI

## What it does

A Swing-based GUI application for drawing and managing shapes (Circle, Rectangle, Square). Includes a canvas for visualization and buttons to add/display different shape types.

## How to run

Compile and run:
```bash
javac *.java
java GUI
```

Or use ShapeTester/PictureTester for other testing modes.

## Dependencies

- Java with Swing (javax.swing, java.awt)
- Shape.java - Abstract shape class
- Circle.java, Rectangle.java, Square.java - Shape implementations
- Canvas.java - Drawing surface
- Layout.java - Main GUI frame
